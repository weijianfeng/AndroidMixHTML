# AndroidMixHTML
android和web，html混合编程，native程序与html代码之间的交互，使用JS作为桥梁。
    
    参考了如下github大神十分有名的jsbridge的代码，迁移到了Eclipse环境上,原工程只有AS版本。
    做些一些小的代码调整，方便自己后续查阅方便
     
[参考lzyzsd/JsBridge](https://github.com/lzyzsd/JsBridge)<br />
